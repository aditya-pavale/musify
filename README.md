# Musify

Musify is a Streamlit-based interactive web application designed to showcase
legendary singers through a clean, immersive, and visually focused interface.
The project emphasizes user experience, interface design, and structured
content presentation rather than algorithmic recommendation systems.

## Overview
The application provides a calm and engaging way for users to explore iconic
artists, their musical journeys, and curated media links. Navigation is handled
through session state, enabling smooth transitions between sections.

## Features
- Custom user interface using CSS
- Artist-based navigation and content display
- Clean layout with background imagery
- Modular code structure for clarity and maintainability

## Technologies Used
- Python
- Streamlit
- HTML/CSS (custom styling)

## Project Structure
- `app.py` – main application logic
- `singers.py` – artist data and metadata
- `styles.css` – custom UI styling
- `icon.png` – application branding asset
